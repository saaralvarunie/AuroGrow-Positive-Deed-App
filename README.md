# AuroGrow-Positive-Deed-App
A gamified habit-tracking Swift Playground app where users complete daily tasks to grow a virtual bird companion. Built for the Swift Student Challenge to promote positivity, consistency, and personal growth through interactive experiences.

## App Overview

- **App Name:** AuroGrow  
- **Domain:** Productivity / Wellness / Gamified Self-Development  
- **Target Audience:** Students, young users, and anyone looking to build positive habits  
- **Platform:** Swift Playgrounds (SwiftPM format)  

---

## Features

- **Personalized Onboarding**  
  Users name their bird and themselves, setting up a personal connection with their virtual companion.

- **Daily Deeds**  
  The app generates a daily list of small positive tasks. Completing them helps the bird grow through visual stages.

- **Pet Progression**  
  Each pet (bird) evolves through stages. Once fully grown, a new pet is introduced automatically, encouraging ongoing habit-building.

- **Achievements & Badges**  
  Users earn badges after completing milestones (e.g., 5 tasks = 1 badge). A progress tracker is included.

- **Profile Customization**  
  Users can edit their name, pet name, and choose from different bird companions. Profile picture upload is supported.

- **Animations & Theming**  
  Green gradient background, confetti animations, and smooth transitions enhance the user experience.

---

## Built With

- **SwiftUI** – for modern declarative UI development  
- **Swift Playgrounds (SwiftPM)** – for submission to the Swift Student Challenge  
- **AppStorage & UserDefaults** – for persistent user data  
- **UIKit Integration** – to support profile image selection  
- **AI Assistance (ChatGPT)** – Used for idea generation, onboarding flow design, animation effects, and code optimization.
